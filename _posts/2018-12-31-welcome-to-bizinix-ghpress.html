The Bizinix GHPress Default Bootstrap 3 Template is the base template from which many other Bizinix GHPress templates are derived.

Getting Started

To use this template with Bizinix GHPress, simply do the following:

  1. Fork the bizinix-cms1/bootstrap3-template repository on GitHub.
  2. Rename your repository to something that describes your site.
  3. Enable GitHub Pages
    • Set the source to the master branch.
    • Do not select a theme.
    • For now, do not choose a custom domain (you can do that later).
    • Make sure "Enforce HTTPS" is checked.
